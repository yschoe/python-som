# python-som

Simple Self-Organizing Map (SOM) implementations and experiments in Python.

## What This Repo Contains

- A 2D-grid SOM implementation (original notebook and script form).
- A 1D-chain SOM variant (notebook and script form).
- Visualization utilities, including post-training animation in script versions.

## File Provenance

- `som.ipynb`: original version written by a human (Yoonsuck Choe), then slightly modified.
- `som.py`: generated by ChatGPT Codex from notebook-style source and updated during this repo work.
- `som1D.ipynb`: generated by ChatGPT Codex as a 1D SOM adaptation.
- `som1D.py`: generated by ChatGPT Codex from `som1D.ipynb` using the same notebook-to-script convention.

## Small Modification in `som.ipynb`

The original notebook was kept intact except for a small robustness fix:

- In `som_bmu`, BMU row/column decoding no longer depends on a global `n`.
- It now derives the column count from the weight tensor shape (`W0.shape[1]`), avoiding global-state coupling.
